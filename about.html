The Project
What Is
PRISMA?

PRISMA is a dynamic image gallery that allows users to explore a curated collection of stunning photographs across multiple categories — nature, automobiles, and animals.

Every interaction is thoughtfully crafted: images open in a fullscreen lightbox, users can navigate with keyboard arrows, and category filters make discovery effortless. The gallery is built to be fast, accessible, and beautiful on every device.

This project was created as a demonstration of modern frontend techniques using only HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies.

✦
What It Does
Key Features
01
🔍

Lightbox Preview

Click any image to open a fullscreen lightbox view with smooth fade animation and complete focus on the photo.

02
⌨️

Keyboard Navigation

Navigate between images using arrow keys. Press Escape to close the lightbox — intuitive and accessible.

03
🗂️

Category Filters

Filter images by Nature, Cars, or Animals with smooth show/hide transitions. Gallery updates instantly.

04
📱

Fully Responsive

Built with CSS Grid. Layout adapts from 4 columns on desktop to 2 on tablet to 1 on mobile seamlessly.

05
✨

Hover Animations

Smooth scale and brightness transitions on every image card with an elegant overlay showing category and a fullscreen icon.

06
🖼️

Dynamic Layout

CSS Grid masonry-style layout with a featured first card that spans two columns for visual drama and hierarchy.
